sdc — Sound Device Control

Standard interface for managing sound capture hardware on Raspberry Pi.

  1. Home
  2. Sound Device Control

Overview

For deployments of Raspberry Pi based acoustic sensors with multiple different sound capture hardware, sdc provides a standard interface for sound device control as well as a standardised installation script.

This software simplifies device setup and control for any number of Raspberry Pi devices, although the greatest benefit comes in managing deployments of large, heterogeneous networks.

Key Features

  • Unified interface for diverse sound capture hardware
  • Standardised installation across devices
  • Simplifies management of large heterogeneous networks
  • Supports playback and capture device configuration
