pi-pwr

Power management tools for Raspberry Pi.

  1. Home
  2. pi-pwr

Overview

When operating from solar panels or batteries, reducing unnecessary power consumption can make a Raspberry Pi run more reliably for longer. pi-pwr is a simple command-line tool that allows you to disable USB, WiFi, Bluetooth, HDMI, and the device LEDs when they are not needed.

Key Features

  • Disable USB ports to reduce power draw
  • Turn off WiFi and Bluetooth radios
  • Disable HDMI output
  • Control onboard LEDs
  • Simple command-line interface
  • Ideal for solar and battery-powered deployments
