Overview

ws-node is the core component of WildlifeSystems. It turns a Raspberry Pi into an autonomous biodiversity monitoring node capable of capturing audio, images, and environmental sensor data in the field.

Designed for remote deployments, ws-node coordinates all other WildlifeSystems components — managing power, sound devices, cameras, and sensors — into a single cohesive monitoring unit.

Key Features

  • Autonomous operation on Raspberry Pi hardware
  • Coordinates audio, image, and sensor data capture
  • Designed for remote and off-grid deployments
  • Integrates with other WildlifeSystems tools (pi-pwr, sdc, sensor-control, ws-camerad)
