Install and configure environmental sensors on the Raspberry Pi.

Overview

sensor-control provides a unified interface for installing and configuring environmental sensors on the Raspberry Pi. It simplifies the process of integrating temperature, humidity, pressure, and other environmental sensors into WildlifeSystems monitoring nodes.

Key Features

  • Unified interface for diverse environmental sensors
  • Standardised installation and configuration
  • Designed for field deployment on Raspberry Pi
  • Integrates with the wider WildlifeSystems ecosystem
